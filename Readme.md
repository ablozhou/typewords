# 打字练习
- Author: Andy Zhou <ablozhou@gmail.com>
- Date: 2021.11.08
- source url: https://github.com/ablozhou/typewords
  
小学单词打字练习。
随机给出单词。打对了会给出中文意思，并计正分。打错则扣分。负分则退出。
输入 [/exit] 结束练习。
可以编辑 words.xlsx来增删修改需要学习的单词和类别

# 命令
直接在提示符输入如下指令
```
命令列表：
----------
/help: 相关命令帮助
/exit: 退出
/choose: 选类别,[99]为全部
/next: 下一个类别
----------

选择要练习的类别索引，[99] 为全部:
```
